# Asta Frontend

This folder contains the Stage 2 frontend for the SME compliance co-pilot.

## Run locally

```powershell
cd frontend
npm install
npm run dev
```

## Build

```powershell
cd frontend
npm run build
```

## What this includes

- A branded landing page for Asta
- A visual dashboard preview for SME and CA demos
- Responsive layout and strong product messaging for Stage 2 validation
